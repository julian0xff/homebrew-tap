# Homebrew Tap

Custom [Homebrew](https://brew.sh) tap for [julian0xff](https://github.com/julian0xff) apps.

## Install

```bash
brew tap julian0xff/tap
```

## Available Casks

| Cask | Description |
|------|-------------|
| `dictava` | Local AI-powered voice dictation for macOS |

### Dictava

```bash
brew install --cask julian0xff/tap/dictava
```

Update:

```bash
brew upgrade --cask dictava
```
